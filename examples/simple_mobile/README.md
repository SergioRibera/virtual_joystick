# Features
- Float Joystick
- Expansive area to interact with joystick

> **NOTE:** The color in area is only for make it visible

# Settings of this example
- VirtualJoystickAxis::Both
- VirtualJoystickType::Floating

# Compilation

To compile android projects you can use [cargo-ndk](https://crates.io/crates/cargo-ndk).

See the [Android Section](https://github.com/bevyengine/bevy/tree/main/examples#android) of the bevy example `README.md` for details.
